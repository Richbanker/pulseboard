## 🚀 Features

- 📈 Live updating chart via WebSocket
- 🧠 Zustand for state management
- 🌐 Mocked WebSocket server (`mock-socket`)
- 💅 Responsive UI with TailwindCSS
- 🟢 Connection status indicator

---

## 🛠 Stack

- React + TypeScript
- Zustand
- Recharts
- TailwindCSS
- mock-socket
- Webpack
